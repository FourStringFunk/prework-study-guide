HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • status: checks what brand we are current on
  • checkout -b brand-name: creates a new brand and switches to it
  • git add -A: adds the changed file from the working branch directory
  • git commit -m "WRITE A DESCRIPTION HERE": commits changes to the repository. -m denotes a descriptive message with the commit
  • git pull origin main: syncs local branch with remote branch on GitHub
  • git push origin BRANCHNAME: pushes the changes to Git
